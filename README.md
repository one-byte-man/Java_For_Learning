# Java_For_Learning ☕

This repository documents my progress in **Java SE 21** and **Object-Oriented Programming (OOP)** principles, following the academic curriculum from Deitel & Deitel.

## 🚀 Purpose

- **Competency Proof:** Technical evidence of my Java and OOP knowledge.
- **Modern Java:** Development using **Java SE 21 (LTS)**.
- **IDE Experience:** Utilizing **Eclipse IDE** to diversify development environment skills.
- **Foundational Strength:** Building a solid base for advanced Unity/C# development through Java's strict OOP structure.

## 📚 Completed Lessons & Topics

### 🏗️ Fundamentals (Lessons 1-5)

- [x] **Lesson 1-3: Basics & I/O**
  - Variables, Memory Concepts, and Primitives.
  - User input handling with `Scanner`.
  - Formatted output using `printf`.
- [x] **Lesson 4: Control Flow (Selection)**
  - Conditional logic: `if`, `else if`, `else`.
  - Shorthand coding: **Ternary Operator** (`? :`).
  - **Modern Switch Expressions:** Java 12+ features (`->` arrow, multiple labels).
- [x] **Lesson 5: Control Flow (Iteration)**
  - `for`, `while`, and `do-while` loops.
  - Loop control keywords: `break` and `continue`.
  - Nested loops and algorithmic pattern exercises.

### 🧱 Core Data Structures & Error Handling

- [x] **Lesson 9: Arrays**
  - Fixed-size arrays, `length`, and literal initialization.
  - Default values, bounds, and reference-copy behavior.
  - `SecureRandom`-based grade distribution and histogram output.
- [x] **Lesson 10: Exception Handling**
  - `try`, `catch`, and `finally` flow control.
  - Specific exception handling: `InputMismatchException`, `ArrayIndexOutOfBoundsException`, and `ArithmeticException`.
  - Cleanup patterns and safe resource closing.

### 🛠️ Advanced Logic & Utility (Lessons 7-8)

- [x] **Lesson 7: Char & Number Mapping**
  - Unicode/ASCII relationships (A-Z: 65-90, a-z: 97-122).
  - Character arithmetic (e.g., `'A' + 32` for lowercase conversion).
- [x] **Lesson 8: Java API & Math Class**
  - `java.lang.Math`: `abs`, `pow`, `sqrt`, `max/min`.
  - Rounding functions: `ceil`, `floor`, `round`.
  - **Trigonometry:** Working with Radians (`toRadians`).
  - Core Packages Overview: `java.lang`, `java.util`, `java.security`.

### 💎 Object-Oriented Programming (OOP1)

- [x] **Lesson 6: Classes & Method Anatomy**
  - **Access Modifiers:** `public`, `private`, `protected`, and `default`.
  - **Static Keyword:** Class-level vs. Instance-level members.
  - **Method Anatomy:** `void` vs `return` types, parameters, and pass-by-value logic.
- [x] **Encapsulation:** Protecting fields with `private` and providing access via getters/setters.
- [x] **Constructors:** Default vs. Parameterized constructors and **Constructor Overloading**.
- [x] **The `this` Keyword:** Eliminating shadowing between fields and parameters.

### 🃏 Object-Oriented Programming (OOP2)

- [x] **Card Class:** Encapsulating card face and suit data with `toString()` support.
- [x] **DeckOfCard:** Building a 52-card deck, shuffling with `SecureRandom`, and dealing cards.
- [x] **TestCards:** Deck initialization and formatted card output in rows.
